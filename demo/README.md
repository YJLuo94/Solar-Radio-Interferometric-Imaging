# Demo Workflows

This directory contains example workflow descriptions and configuration templates for MeerKAT and VLA solar data.

- `example_meerkat_config.yml`: example configuration for MeerKAT solar processing.
- `meerkat_workflow_demo.md`: step-by-step MeerKAT workflow outline.
- `vla_workflow_demo.md`: placeholder VLA workflow outline for future generalization.
