# Workflow Overview

The workflow is organized as a sequence of modular processing stages:

1. Input configuration and Measurement Set preparation.
2. Pre-calibration inspection and flagging.
3. Cross-calibration using standard calibrators.
4. Optional solar self-calibration.
5. Quick-look and science imaging.
6. Primary-beam correction.
7. Science-product generation.

The repository is intended to keep each step transparent and adjustable. This is particularly important for solar radio data, where real solar emission can be strong, variable, and highly structured in both time and frequency.
