# Solar Self-calibration Notes

Solar self-calibration should be treated as an inspection-driven step. Recommended checks include:

- inspect pre-self-calibration images;
- verify that CLEAN masks do not include obvious artifacts;
- solve calibration tables per suitable frequency chunk or SPW;
- inspect phase/amplitude solutions before applying them broadly;
- compare dynamic range and image morphology before and after each round;
- avoid modifying the original Measurement Set directly.
