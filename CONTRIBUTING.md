# Contributing

Contributions are welcome. This repository is intended to document and share a transparent workflow for solar radio data processing. Before submitting changes, please make sure that:

1. Large data products are not committed.
2. Scripts use clear English comments.
3. New processing steps are documented in the relevant module README.
4. Example parameters are safe placeholders and do not depend on private local paths.

For major changes, please open an issue or discussion first.
