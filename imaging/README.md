# Imaging

This module contains quick-look imaging and final science imaging scripts.

- `qlook_image.py`: generate quick-look images for inspection.
- `sci_image.py`: generate science-quality images and optional radio/EUV context plots.

- `full_disk_science_imaging.py`: final full-disk science imaging for self-calibrated MeerKAT data, including Stokes I/V imaging and FITS export.
